Magic Mirror

一面有魔法的镜子。它看得见你，把小朋友和家人变成故事里的主角， 生成属于你们自己的漫画与动画片——陪着孩子的好奇心一起长大。

private beta · 内测中

// 它是什么

👀

看见你，变成角色

通过镜子里的镜头认出小朋友和家人，把每个人变成一个可爱、可复用的故事角色。

🎬

你的故事，你来主演

以你们为主角，生成一格格漫画，再变成一部小小动画片。每个故事都独一无二。

✨

魔法是有限的

魔镜每天只有一点点法力。想做更多作品？靠现实里的好表现，让爸爸妈妈为你充能。

// 为什么

给小朋友的好奇心一个出口。「蝴蝶为什么会飞？」——把问题变成一个属于他的小故事，而不是又一段刷不完的视频。
让"用屏幕"变成"一起创作"。魔法额度天然限制使用时间，也把它变成亲子之间的奖励与陪伴。
把童年留下来。那些角色和故事会沉淀成一份可以回看的成长档案。
